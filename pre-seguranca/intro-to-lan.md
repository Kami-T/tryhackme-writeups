# Intro to LAN

- **Trilha:** Pré Segurança — Módulo de Redes
- **Data:** 01/07/2026

## O que aprendi
Como redes locais (LAN) são organizadas e o conceito de subnetting
(dividir uma rede em sub-redes menores). Os três usos do IP numa
sub-rede: network address (início da rede), host address (identifica
dispositivo) e default gateway (envia dados para outra rede).

## Conexão com segurança
Subnetting é uma defesa: segmentar a rede isola sistemas e contém
ataques (ex: separar rede de funcionários da rede pública de um café).
ARP spoofing é um ataque man-in-the-middle a monitorar no SOC.

# Data Representation

- **Trilha:** Pré Segurança — Módulo 4
- **Data:** 28/06/2026

## O que aprendi
Como os dados são representados e convertidos: binário (base 2),
hexadecimal (base 16) e a relação entre eles (cada dígito hex = 4 bits).
Pratiquei conversões entre hex, binário e decimal.

## Conexão com segurança
- Base64 é muito usado por atacantes para ofuscar comandos e payloads
  (phishing, scripts de PowerShell).
- Hexadecimal é base para análise de malware e forense (magic bytes /
  assinaturas de arquivo).

## Ferramenta pra lembrar
CyberChef — ferramenta web para conversão e decodificação (hex, Base64,
binário etc.), muito usada em SOC.

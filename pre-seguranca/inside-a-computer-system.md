# Inside a Computer System

- **Trilha:** Pré Segurança
- **Data:** 26/06/2026

## O que aprendi
Como um sistema computacional funciona por dentro: o papel do sistema
operacional, hardware (CPU, RAM, armazenamento) e como software e
hardware se comunicam. Base importante para entender análise de
processos e detecção de ameaças em endpoints mais à frente.

## Conceitos novos
Operating System, kernel, CPU, RAM, storage, processos.

# Python: Simple Demo

- **Trilha:** Pré Segurança — Módulo 4
- **Data:** 30/06/2026

## O que aprendi
Sintaxe básica do Python na prática, construindo um jogo de adivinhação:
input/output, conversão de tipos (int(), input()), condicionais com
if / elif / else e a lógica de comparar o palpite com o número secreto.

## Conexão com dev / segurança
A lógica é igual ao if/else if/else do JavaScript — muda só a sintaxe
(elif, dois-pontos, indentação no lugar das chaves). Python é a
linguagem da automação em cibersegurança: processar logs, consultar
APIs de threat intel, automatizar tarefas repetitivas no SOC.

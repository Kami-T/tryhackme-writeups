- **Trilha:** Pré Segurança — Módulo 4

## O que aprendi
Revisão de JavaScript no Node construindo o jogo de adivinhação:
console.log() para exibir, rl.question() para input, parseInt() para
converter string em número, loop while e condicionais if/else if/else.
Operador !== (estritamente diferente).

## Conexão com segurança
JavaScript é central em segurança web: base do XSS (Cross-Site
Scripting), roda client-side (nunca confiar em validação só no front),
e aparece ofuscado em phishing/malware web. Minha base de dev vira
diferencial pra AppSec (meu objetivo de longo prazo).

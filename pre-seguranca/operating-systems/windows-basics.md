# Windows Basics

- **Trilha:** Pré Segurança — Módulo 3 (Operating Systems Basics)
- **Data:** 28/06/2026

## O que aprendi
Naveguei pelo sistema Windows: Configurações, região/idioma, e o
Gerenciador de Tarefas (contas de usuário e processos). Explorei o
Windows Security e investiguei o histórico de ameaças (Threat history).

## Destaque — investigação de ameaça
Analisei um alerta do Windows Defender: um arquivo de teste EICAR
detectado e colocado em quarentena. Identifiquei o arquivo afetado
(tryhatmemaldoc.txt) e seu caminho no sistema. O EICAR é um arquivo
padrão da indústria, inofensivo, usado para testar antivírus.

## Conceitos novos
Windows Security, Threat history, quarentena, EICAR, Task Manager,
investigação de ativos.

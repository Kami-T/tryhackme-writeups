# Operating Systems: Introduction

   - **Trilha:** Pré Segurança — Módulo 3 (Operating Systems Basics)
   - **Data:** 27/06/2026

   ## O que aprendi
   Os conceitos básicos de sistemas operacionais. O kernel é o núcleo do
   SO e opera no "kernel space", com acesso irrestrito ao hardware. O SO
   também cuida do "user management" (gestão de contas, autenticação e
   permissões). Explorei uma máquina Linux (Ubuntu/MATE) via System Monitor.

   ## Conceitos novos
   Kernel space, user space, user management, System Monitor, recursos do SO.
